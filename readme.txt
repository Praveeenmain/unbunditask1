# Bootstrap Image Carousel

This project demonstrates how to create a simple image carousel using Bootstrap 4. It showcases a set of images in a slideshow format, allowing easy navigation between images.

## Features

- Presents a collection of images in a carousel format.
- Utilizes Bootstrap 4 components for a responsive and interactive carousel.
-I used the Popper.js - JavaScript library used for positioning popovers, tooltips, dropdowns, and other UI elements. It's utilized by Bootstrap for handling element positioning.
-I used Bootstrap JavaScript-It includes all the Bootstrap components and functionalities, such as modals, carousels, tooltips, and more. It's necessary to include this script if you want to use Bootstrap's JavaScript-based features.
